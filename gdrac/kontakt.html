Autor

Aplikacja powstałą na potrzeby projektu "Esri Global Disaster Resilience App Challenge 2014". Celem jest przedstawienie funkcjonalności platformy ArcGIS. Aplikacja może być wykorzystywana jedynie do celów demonstracyjnych.

Technologia

Aplikacja używa:
- Bootstrap
- Esri JavaScript API
- Usług mapowych, obiektów i geoprzetwarzania ArcGIS Online
